üöÄ Deployment Steps
Step 1: Push to Git Repository
git add .git commit -m "Add Render deployment configuration"git push origin main
Step 2: Deploy via Render Dashboard
Go to https://dashboard.render.com
Click "New +" ‚Üí "Blueprint"
Connect your GitHub/GitLab repository
Select your ai-writing-assistant repository
Render will auto-detect the render.yaml and show both services
Click "Apply" to deploy
Step 3: Configure Environment Variables
After the initial deployment:
Go to your Backend service ‚Üí Environment
Set the AI provider you want to use:
Variable	Value	Description
AI_PROVIDER	demo / groq / gemini / openai	Choose your AI provider
GROQ_API_KEY	your-key	FREE - Get at console.groq.com
GEMINI_API_KEY	your-key	FREE - Get at aistudio.google.com
OPENAI_API_KEY	your-key	Paid - Get at platform.openai.com
Step 4: Update Frontend API URL
After backend deploys, you'll get a URL like https://ai-writing-assistant-api.onrender.com
Update frontend/src/environments/environment.prod.ts:
export const environment = {  production: true,  apiUrl: 'https://ai-writing-assistant-api.onrender.com/api'  // Your actual backend URL};
Then rebuild:
git add .git commit -m "Update production API URL"git push
Render will auto-redeploy the frontend.
